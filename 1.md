# ## **SECTION 1 â€” THEORY QUESTIONS WITH ANSWERS**

---

### **1. What is the difference between variable declaration and definition?**

* **Declaration:** Introduces variable name and type (no memory allocated).
* **Definition:** Allocates memory to variable.

> In C, declaration = definition unless `extern` is used.

---

### **2. What are keywords? Give examples.**

Reserved words with predefined meaning.
Examples: `int`, `char`, `if`, `else`, `for`, `while`, `return`.

---

### **3. List four data types in C.**

`int`, `char`, `float`, `double`.

---

### **4. What is the use of sizeof operator?**

Returns number of bytes used by a data type/variable.

---

### **5. Explain operator precedence.**

Defines the order in which operators are evaluated in an expression.

---

### **6. What is recursion?**

A function calling itself.

---

### **7. What is an array?**

A collection of elements of the same type stored in contiguous memory.

---

### **8. What is a multidimensional array?**

Array containing other arrays. Example:

```c
int matrix[3][3];
```

---

### **9. What is pass-by-value?**

Function receives a **copy** of the argument. Changes do not affect original variable.

---

### **10. What is pass-by-reference?**

Function receives the **address** of the variable. Changes affect the original variable.

    int i=0, j=strlen(s)-1;
    while(i<j){

        i++; j--;
    }
    return 1;
}
